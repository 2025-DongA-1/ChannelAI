# 대시보드 UX 개선 가이드

## 🎯 개선 목표
마케팅 지식이 전혀 없는 소상공인도 한눈에 광고 성과를 이해할 수 있도록 직관적인 대시보드 구축

## ✨ 주요 개선 사항

### 1. 스마트 색상 코딩 시스템
광고 성과 지표(CTR, CPC, ROAS)에 대해 자동으로 색상을 부여하여 좋고 나쁨을 직관적으로 표시합니다.

**색상 기준:**
- 🟢 **녹색 (좋음)**: 업종 평균 대비 우수한 성과
- 🟡 **노란색 (보통)**: 업종 평균 수준의 성과  
- 🔴 **빨간색 (개선 필요)**: 업종 평균 미달, 개선이 필요함

### 2. 지표별 벤치마크 기준

#### CTR (클릭률)
- **좋음**: 3.5% 이상
- **보통**: 2.0% ~ 3.5%
- **나쁨**: 2.0% 미만

#### CPC (클릭당 비용)
- **좋음**: 500원 이하
- **보통**: 500원 ~ 1,000원
- **나쁨**: 1,000원 초과

#### ROAS (광고 수익률)
- **좋음**: 4.0배 이상
- **보통**: 2.5배 ~ 4.0배
- **나쁨**: 2.5배 미만

### 3. AI 기반 전략 조언
각 지표에 마우스를 올리면 현재 성과에 맞는 구체적인 조언이 표시됩니다.

**조언 예시:**
- **CTR 좋음**: "훌륭합니다! 광고 소재가 타겟에게 매력적으로 다가가고 있어요. 현재 전략을 유지하세요."
- **CPC 높음**: "비용이 높아요. 경쟁이 낮은 키워드를 찾거나, 타겟 범위를 조정해보세요."
- **ROAS 낮음**: "수익성 개선이 필요해요. 광고 대상 상품이나 서비스를 재검토해보세요."

### 4. 시각적 개선 요소

#### 상태 배지
- 각 지표 카드 상단에 실시간 상태 배지 표시
- 애니메이션 효과로 시각적 주목도 향상

#### 진행도 표시
- 미니 진행 바로 현재 값이 기준 범위 내 어디에 위치하는지 표시
- 한눈에 개선 여지를 파악 가능

#### 호버 툴팁
- 마우스 오버 시 자세한 설명과 조언 표시
- "마우스를 올려보세요 👆" 힌트로 사용자 유도

## 🎨 사용자 경험 설계 원칙

### 1. 단순성 (Simplicity)
- 전문 용어 최소화
- 평이한 한국어 표현 사용
- "클릭률"보다 "광고를 본 사람 중 클릭한 비율" 같은 설명적 표현

### 2. 즉시성 (Immediacy)
- 색상만 봐도 좋고 나쁨을 즉시 파악
- 복잡한 계산 없이 시각적으로 이해

### 3. 실행 가능성 (Actionability)
- 단순히 데이터만 보여주지 않음
- 각 상황에 맞는 구체적인 행동 제안
- "무엇을 해야 하는지" 명확히 제시

### 4. 맥락 제공 (Context)
- 절대값뿐만 아니라 상대적 위치 표시
- 업종 평균과의 비교를 통한 벤치마킹

## 📊 향후 개선 방향

### 단기 (1-2주)
1. ✅ 색상 코딩 시스템 적용
2. ✅ 호버 툴팁으로 전략 조언 제공
3. [ ] 실제 업종별 벤치마크 데이터 수집

### 중기 (1개월)
1. [ ] 사용자의 과거 성과 추이 시각화
2. [ ] 개선 제안의 우선순위 표시
3. [ ] 간단한 "시작하기" 가이드 추가

### 장기 (3개월)
1. [ ] AI 모델을 활용한 개인화된 벤치마크
2. [ ] 자동 알림 시스템 (성과 급락 시)
3. [ ] 성공 사례 및 팁 제공

## 🔧 기술 구현

### 프론트엔드 컴포넌트
- `PerformanceMetricCard`: 색상 코딩과 툴팁이 적용된 지표 카드
- 동적 색상 계산 로직
- CSS 애니메이션 효과
